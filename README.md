# Nutrition & Health Calculator Platform

### Full-Stack Web Application | Digital Product for Football Players  
A nutrition and health calculator platform designed to generate personalized nutritional insights, dashboard results, and downloadable PDF reports through a secure access-code flow.

---

## Overview

The **Nutrition & Health Calculator Platform** is a web-based digital product created for Mosabri’s football audience.  
It helps users calculate and understand key nutrition values such as calories, macros, BMI-related insights, and personalized health indicators through a simple Arabic user interface.

The platform was designed as a paid digital product, where users access the calculator using an email-based access code, complete the required inputs, view their results through a dashboard, and generate a downloadable PDF report.

This repository is presented as a **project showcase** because the full source code contains sensitive configuration files, access logic, environment setup, and internal implementation details.

---

## Project Purpose

The main goal of this project is to transform sports nutrition knowledge into a practical digital product that users can access and benefit from directly.

Instead of providing static content only, the platform creates a more interactive experience by allowing users to:

- Enter their personal information
- Generate nutrition-related values
- Review the results through dashboard screens
- Export the results as a PDF report
- Access the product using a controlled access-code system

## Key Features

| Feature | Description |
|---|---|
| **Nutrition Calculator** | Calculates personalized nutrition-related values based on user inputs. |
| **Arabic RTL Interface** | Designed with Arabic users in mind using a right-to-left interface experience. |
| **Email-Based Access Code** | Uses a controlled access-code flow generated based on the user’s email. |
| **Dashboard Experience** | Displays user results and nutrition values in structured dashboard screens. |
| **PDF Report Generation** | Allows users to generate and download a personalized PDF nutrition report. |
| **Admin Login** | Includes an admin login screen for internal access and management. |
| **Digital Product Flow** | Designed to support a paid product experience connected to Mosabri’s audience and sales pages. |

## Product Flow

| Step | Description |
|---|---|
| **1. Access** | User enters through the login/access screen using an email-based access code. |
| **2. Calculator Input** | User fills in the required nutrition and body-related information. |
| **3. Processing** | The platform calculates the nutrition values based on the user’s inputs. |
| **4. Dashboard** | User reviews the generated values through organized dashboard screens. |
| **5. PDF Report** | User generates and downloads a personalized nutrition report. |
| **6. Admin Access** | Admin can access internal product features through a dedicated login screen. |
